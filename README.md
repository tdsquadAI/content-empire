## 📂 Repository Structure

```
content-empire/
├── site/                    # Hugo static site
│   ├── content/
│   │   ├── posts/           # Blog articles
│   │   ├── courses/         # Course modules
│   │   └── pages/           # Static pages (about, etc.)
│   ├── layouts/             # Hugo templates
│   └── static/              # CSS, images
├── brand/                   # Brand guide and assets
├── content-calendar/        # Publishing schedule
├── social/                  # Social media profiles & bios
└── docs/                    # Built site (GitHub Pages)
```

## 🚀 Running Locally

```bash
# Install Hugo (https://gohugo.io/installation/)
cd site
hugo server -D

# Open http://localhost:1313
```

## 🤝 Contributing

We welcome contributions! Here's how:

1. **Write an article** — Fork the repo, add a markdown file to `site/content/posts/`, and open a PR
2. **Fix a bug** — Found an issue with the site? Open an issue or PR
3. **Suggest a topic** — Open an issue with the "content-idea" label
